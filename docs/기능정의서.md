# FitLife Pro 기능 정의서 (Feature Specification)

| 항목 | 내용 |
| :--- | :--- |
| **프로젝트명** | FitLife Pro |
| **버전** | 1.0 |
| **작성일** | 2025-12-12 |
| **플랫폼** | 모바일 웹 앱 (Responsive Web App) |
| **기술 스택** | React, Tailwind CSS, Firebase (Auth, Firestore) |

---

## 1. 개요 (Overview)
FitLife Pro는 사용자가 운동 루틴을 계획하고, 실시간으로 수행 기록을 측정하며, 그 결과를 커뮤니티 피드와 랭킹 시스템을 통해 공유하여 동기부여를 얻는 **올인원 피트니스 플랫폼**입니다.

## 3. 데이터 구조 (Data Schema - Firestore)

### 3.1. Users Collection
* **Path:** `/artifacts/{appId}/users/{userId}`
    * `profile/info`: `uid`, `name`, `email`, `provider`, `createdAt`
    * `data/stats`: `totalVolume`, `weeklyVolume`, `workoutsCompleted`, `weeklyActivity` (Array)
    * `data/currentRoutine`: `routine` (Array of Exercise Objects)

### 3.2. Public Feeds Collection
* **Path:** `/artifacts/{appId}/public/data/feeds`
    * **Document Structure:**
        * `uid`: 작성자 ID
        * `user`: 작성자 이름
        * `content`: 게시글 내용
        * `stats`: `{ volume, time }`
        * `likes`: 좋아요 수 (Number)
        * `likedBy`: 좋아요 누른 유저 ID 배열 (Array)
        * `createdAt`: 생성 일시 (Timestamp)

## 4. UI/UX 디자인 가이드

* **테마:** 다크 모드 (`Neutral-900` 배경) + 네온 옐로우 (`Yellow-400`) 포인트 컬러.
* **인터랙션:**
    * 운동 목록 드래그 앤 드롭 시 부드러운 위치 변경.
    * 운동 타이머 작동 시 SVG 원형 프로그레스 바 애니메이션.
    * 버튼 클릭 및 탭 전환 시 즉각적인 반응형 피드백.
    * 중요 액션(종료, 로그아웃) 시 모달을 통한 재확인.
    * 작업 완료 시 Toast 메시지 노출.
